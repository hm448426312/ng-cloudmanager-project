输入

{{{
    url: '/IAM/v1/user/impUser',
    method: 'POST',
    autoUpload: true
  } | json}}

options: 类型如上; desc: url:文件上传的接口/method:接口请求类型/autoUpload:是否自动上传

fileMaxSize: number; eg: 1024; desc: 支持文件最大（单位B）

acceptType: Array; eg: {{['.png'] | json}}; desc: 支持上传的文件类型，暂时支持的类型有：.jpg,.jpeg,.gif,.csv,.doc,.png,.ppt,.wps,.xls,.xml,.zip,.xlsx,.xltx,.pptx,.docx

multiple: Boolean; eg: true | false; desc: 是否支持多文件上传(暂不支持)

showList: Boolean; eg: true | false; desc: 是否显示上传列表（暂不支持）

输出

fileChangeEvent: 返回数据格式 File; desc: 选中文件回调方法(可以忽略，因为自动上传的)

successItemdesc: 上传成功后回调方法

errorItem: 'overSize'（文件过大） | 'errorType'（类型错误）desc: 上传失败后回调方法
